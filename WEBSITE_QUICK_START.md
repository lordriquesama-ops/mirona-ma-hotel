# Hotel Website - Quick Start

## 🌐 Access Your Website

**Public Website**: `http://localhost:5173/website`
**Admin Dashboard**: `http://localhost:5173/`

## ✨ What's New

Your hotel now has a professional public website with:
- ✅ Real-time room availability checking
- ✅ Automatic room assignment from available rooms
- ✅ Online booking system
- ✅ Email confirmations
- ✅ WhatsApp integration
- ✅ Full dashboard integration

## 🎯 Quick Test

1. Open `http://localhost:5173/website`
2. Click "Check Availability" section
3. Select check-in and check-out dates
4. See available rooms for each category
5. Click "Book This Room" on any category
6. Fill in the form and submit
7. Check your email for confirmation
8. View the booking in admin dashboard

## 🔑 Key Features

### For Guests:
- Browse room categories with prices
- Check real-time availability
- See total cost before booking
- Get automatic room assignment
- Receive email confirmation
- Optional WhatsApp confirmation

### For You (Hotel Staff):
- All bookings appear in dashboard
- Room is automatically assigned
- No double-booking possible
- Guest profiles created automatically
- Full control from admin panel

## 📋 How Availability Works

1. Guest selects dates
2. System checks all bookings
3. Shows available rooms per category
4. Guest books a category (not specific room)
5. System assigns first available room
6. Confirmation sent with room number

## 🎨 Customize Your Website

### Update Content:
1. Go to admin dashboard
2. Click "Website CMS" tab
3. Edit hero title, about text, etc.
4. Save changes

### Update Hotel Info:
1. Go to "Settings" tab
2. Update name, phone, email
3. Changes reflect on website immediately

### Manage Rooms:
1. Go to "Rooms" tab
2. Add/edit room categories
3. Set prices and availability
4. Website updates automatically

## 💡 Pro Tips

- **Availability Badge**: Green badge shows available rooms, red shows fully booked
- **Price Calculator**: Total price updates automatically when dates change
- **Room Assignment**: System picks first available room in category
- **No Payment**: Guests pay on arrival, no online payment needed
- **Email Confirmation**: Sent automatically to guest's email
- **WhatsApp Option**: Guest can send details via WhatsApp for faster response

## 🚨 Important Notes

- Bookings are created with "PENDING" status
- Staff should confirm bookings from dashboard
- Room is assigned automatically (no guest selection)
- System prevents double-booking automatically
- Maintenance rooms are excluded from availability

## 📊 Booking Flow

```
Guest Visits Website
    ↓
Selects Dates
    ↓
Checks Availability
    ↓
Chooses Room Category
    ↓
Fills Booking Form
    ↓
System Assigns Available Room
    ↓

    ↓
Email Confirmation Sent
    ↓

```
